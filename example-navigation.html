Navigation Components

Navigation components help users move through your application. The GitHub Theme provides clean, accessible navigation styles.

Available Classes

  • .nav - Base navigation container
  • .nav-item - Individual navigation item
  • .nav-link - Navigation link styling
  • .nav-link.active - Active/current page indicator

Usage

<ul class="nav">
  <li class="nav-item">
    <a href="#" class="nav-link active">Home</a>
  </li>
  <li class="nav-item">
    <a href="#" class="nav-link">About</a>
  </li>
</ul>
