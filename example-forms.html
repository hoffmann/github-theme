Form Components

Forms are essential for user input and interaction. The GitHub Theme provides styled form controls that are consistent, accessible, and easy to use.

Available Classes

  • .form-group - Container for form fields with proper spacing
  • .form-label - Styled label for form inputs
  • .form-control - Base input styling for text, email, textarea, etc.

Usage

<div class="form-group">
  <label class="form-label" for="email">Email</label>
  <input type="email" class="form-control" id="email" placeholder="your@email.com">
</div>

Best Practices

  • Always use <label> elements for accessibility
  • Associate labels with inputs using the for and id attributes
  • Provide helpful placeholder text
  • Use appropriate input types (email, password, number, etc.)
  • Include validation feedback when necessary
