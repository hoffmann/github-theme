Grid System Documentation

Grid System

The GitHub Theme includes a powerful 12-column responsive grid system that makes it easy to create layouts for any screen size.

How It Works

The grid system is based on three main components:

  • Container: Wraps your content and provides proper padding (.container)
  • Row: Creates a horizontal group of columns (.row)
  • Columns: Your content goes inside columns (.col-*)

Basic Structure

<div class="container">
  <div class="row">
    <div class="col-6">Column 1 (50%)</div>
    <div class="col-6">Column 2 (50%)</div>
  </div>
</div>

The 12-Column System

Each row is divided into 12 equal parts. You can combine columns to create different widths:

  • .col-12 = 100% width (12/12)
  • .col-6 = 50% width (6/12)
  • .col-4 = 33.33% width (4/12)
  • .col-3 = 25% width (3/12)
  • And so on...

Responsive Breakpoints

The grid includes three breakpoints for different screen sizes:

  • Small (sm): ≥576px (landscape phones)
  • Medium (md): ≥768px (tablets)
  • Large (lg): ≥1024px (desktops)

Mobile-First: Classes apply to their breakpoint and all larger screens. For example, .col-md-6 applies to medium, large, and extra-large screens.

Common Patterns

Pattern Classes Behavior

Best Practices

  • Always use a .row: Columns must be direct children of a .row element.
  • Don't exceed 12 columns: Each row should add up to 12 or less (remaining space is left empty).
  • Mobile-first thinking: Start with .col-12 for mobile, then add responsive classes.
  • Use containers: Wrap rows in .container for proper centering and max-width.
  • Combine breakpoints: Use multiple classes like .col-12 .col-sm-6 .col-lg-4 for complex responsive behavior.
  • Test on real devices: Always check your layouts on actual mobile devices, not just browser resize.

Tips

  • Nested Grids: You can nest rows inside columns to create more complex layouts.
  • Gutter Space: Columns have built-in padding (8px on each side) to create gutters between them.
  • Centering: Use fewer than 12 columns to center content (e.g., .col-8 will be centered with space on both sides).
  • Combine with Utilities: Mix grid classes with utility classes like .mt-4 or .text-center for fine-tuned control.
