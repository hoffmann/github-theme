Utility Classes

Utility Classes

Utility classes provide quick, single-purpose styling options that can be applied directly to elements. They're perfect for fine-tuning layouts without writing custom CSS.

Categories

  • Spacing - Margin and padding utilities
  • Text - Alignment and color utilities
  • Display - Control element display properties
  • Flexbox - Flexbox layout utilities (see Flexbox Examples)

Spacing Scale

All spacing utilities use an 8px base unit:

  • *-0 = 0px
  • *-1 = 8px
  • *-2 = 16px
  • *-3 = 24px
  • *-4 = 32px
  • *-5 = 40px

Responsive Display

Display utilities can be combined with responsive breakpoints:

  • .d-sm-* - Applies at 576px and up
  • .d-md-* - Applies at 768px and up
  • .d-lg-* - Applies at 1024px and up
Example: .d-none .d-md-block hides on mobile, shows on tablet and up.
