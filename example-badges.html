Badge Components

Badges are small count and labeling components. They're perfect for displaying status, counts, categories, and labels.

Available Classes

  • .badge - Base badge class (required)
  • .badge-primary - Primary blue badge
  • .badge-success - Success green badge
  • .badge-danger - Danger red badge
  • .badge-secondary - Secondary gray badge

Usage

<span class="badge badge-primary">New</span>
<span class="badge badge-success">Active</span>
<span class="badge badge-danger">5</span>

Use Cases

  • Status indicators (Active, Inactive, Pending)
  • Notification counts
  • Category labels
  • Version numbers
  • Feature tags
