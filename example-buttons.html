Button Components

Buttons are essential UI elements for user interactions. The GitHub Theme provides multiple button variants, sizes, and states to suit different needs.

Available Classes

  • .btn - Base button class (required)
  • .btn-primary - Primary action button
  • .btn-secondary - Secondary action button
  • .btn-success - Success/confirmation button
  • .btn-danger - Destructive/danger button
  • .btn-outline - Outline style button
  • .btn-sm - Small button
  • .btn-lg - Large button
  • .btn-block - Full-width button

Usage

<button class="btn btn-primary">Click Me</button>
<a href="#" class="btn btn-secondary">Link Button</a>
<button class="btn btn-primary btn-lg">Large Button</button>
