Flexbox Utilities

This page demonstrates all available flexbox utility classes in the GitHub Theme. Flexbox is a powerful layout tool that makes it easy to align and distribute elements in containers.

Available Utility Classes

  • .d-flex - Enables flexbox layout on a container
  • .flex-row - Sets flex direction to row (default, horizontal)
  • .flex-column - Sets flex direction to column (vertical)
  • .justify-content-* - Controls horizontal alignment (main axis)
  • .align-items-* - Controls vertical alignment (cross axis)

How to Use

Simply add the .d-flex class to any container, then combine it with other utility classes to control layout:

<div class="d-flex justify-content-center align-items-center">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

Quick Reference

Class Property Description
.d-flex display: flex Enables flexbox layout
.flex-row flex-direction: row Items flow horizontally (default)
.flex-column flex-direction: column Items flow vertically
.justify-content-start justify-content: flex-start Pack items at the start
.justify-content-end justify-content: flex-end Pack items at the end
.justify-content-center justify-content: center Center items
.justify-content-between justify-content: space-between Distribute items with space between
.justify-content-around justify-content: space-around Distribute items with space around
.align-items-start align-items: flex-start Align items at the start
.align-items-end align-items: flex-end Align items at the end
.align-items-center align-items: center Center align items
.align-items-baseline align-items: baseline Align items by baseline
.align-items-stretch align-items: stretch Stretch items to fill container

Tips

  • Main Axis vs Cross Axis: In flex-row, main axis is horizontal and cross axis is vertical. In flex-column, they are reversed.
  • justify-content: Controls spacing along the main axis (horizontal in row, vertical in column).
  • align-items: Controls alignment along the cross axis (vertical in row, horizontal in column).
  • Combining Classes: You can combine any justify and align classes together for precise control.
  • Responsive: While not shown here, you can combine these with responsive utilities like .d-sm-flex, .d-md-flex, etc.
