Alert Components

Alerts provide contextual feedback messages for user actions. They come in four different types to convey different types of information.

Available Classes

  • .alert - Base alert class (required)
  • .alert-info - Informational messages (blue)
  • .alert-success - Success messages (green)
  • .alert-warning - Warning messages (yellow)
  • .alert-danger - Error/danger messages (red)

Usage

<div class="alert alert-info">
  <strong>Info:</strong> This is an informational message.
</div>

Best Practices

  • Use info alerts for general information and tips
  • Use success alerts for successful operations
  • Use warning alerts for cautionary messages
  • Use danger alerts for errors and critical issues
  • Keep alert messages concise and actionable
  • Include a <strong> label to categorize the alert type

Alert Types

Info: This is an informational alert message.
Success: Your action was completed successfully.
Warning: Please be careful with this action.
Error: Something went wrong. Please try again.
